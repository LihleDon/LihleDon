## What I build

I design and build **data pipelines that actually work in production** — from raw ingestion to clean, queryable, AI-ready data. My focus is on building infrastructure that data teams can rely on: automated, monitored, well-documented, and cost-aware.

I work across the full data engineering stack:
- **Ingestion** — REST APIs, streaming (Kinesis), batch file processing
- **Transformation** — dbt models, PySpark, AWS Glue, Pandas
- **Orchestration** — Apache Airflow DAGs, AWS Step Functions, Lambda
- **Storage** — S3 data lakes, DynamoDB, Athena, DuckDB
- **AI integration** — LLM-powered data quality checks, anomaly detection, automated documentation
